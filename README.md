# Marketplace Eventos

## Descripción
Este proyecto es un monorepo que incluye:
- **Frontend**: Desarrollado en Next.js.
- **Backend**: Desarrollado en NestJS.
- **Mobile**: Desarrollado en Flutter.

## Infraestructura y Despliegue
La infraestructura y los despliegues estarán basados en un VPS (como DigitalOcean). Se ha eliminado la opción de Vercel como principal para el despliegue de la aplicación.

## Orquestación
Utilizamos Docker y docker-compose para la orquestación de nuestros servicios, permitiendo una gestión sencilla y eficiente de los contenedores.

## Estructura de Carpetas
```
marketplace-eventos/
├── frontend/
├── backend/
└── mobile/
```

## Desarrollo Mobile en Flutter
Para el desarrollo en Flutter, se recomienda seguir las siguientes pautas:
- Asegurarse de tener el SDK de Flutter instalado.
- Utilizar un editor compatible, como Visual Studio Code o Android Studio.
- Seguir las mejores prácticas de Flutter para la estructura de la aplicación.
